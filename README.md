## 简介
### 12.6和13.2区别
WiFi驱动不一致，分别使用 AirportItlwm  [v2.2.0-alpha](https://github.com/OpenIntelWireless/itlwm/releases/tag/v2.2.0-alpha)对应的版本。

### common版本
用于在没有独显安装时点亮屏幕。
能正常点亮，但不能流畅使用。无实用性。
